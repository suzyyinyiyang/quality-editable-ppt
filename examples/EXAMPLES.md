# Style Prompts

Available styles:

- 蓝绿产品发布风
- 蓝黑科技运营风
- 紫色科技色块风
- 紫蓝AI科技风
- 深蓝深红攻防风
- 深灰绿色监控风
- 深灰金色年会风
- 自动推荐

`自动推荐` is a router, not a fixed style.
